# Tooltip Component

The Tooltip component displays helpful information when users hover over an element. It accepts any content as a child (icons, text, etc.) and shows the tooltip text above the trigger element.

## Basic Usage

The most common use case is to wrap an icon or text element with helpful context information that appears on hover.

</Tooltip>

```jsx
import { Tooltip } from "astrobits";
```
