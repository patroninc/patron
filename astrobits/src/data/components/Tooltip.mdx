# Tooltip Component

The Tooltip component provides contextual help information that appears on hover. It supports `primary`, `secondary-warm`, and `secondary-neutral` variants with customizable positioning options (top, bottom, left, right).

## Primary

The primary tooltip uses the main theme colors and is ideal for important help information or key feature explanations.

<Tooltip content="This is a primary tooltip with important information" variant="primary">
  Hover me (Primary)
</Tooltip>

```jsx
import { Tooltip } from "astrobits";

<Tooltip content="This is a primary tooltip with important information" variant="primary">
  Hover me (Primary)
</Tooltip>;
```

## Positioning

The tooltip can be positioned on different sides of the trigger element. Available positions are `top` (default), `bottom`, `left`, and `right`.

### Top Position

<Tooltip content="Tooltip appears above the element" position="top">
  Top Tooltip
</Tooltip>

```jsx
<Tooltip content="Tooltip appears above the element" position="top">
  Top Tooltip
</Tooltip>
```

## HTML Content

The tooltip supports HTML content through the `content` prop, allowing for rich formatting within the tooltip.

<Tooltip content="<strong>Bold text</strong> and <em>italic text</em> in tooltip">
  HTML Content Tooltip
</Tooltip>

```jsx
<Tooltip content="<strong>Bold text</strong> and <em>italic text</em> in tooltip">
  HTML Content Tooltip
</Tooltip>
```
